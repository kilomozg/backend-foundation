# Backend Foundation

Практический репозиторий по изучению Python и основ backend-разработки.

## Цель

Сформировать прочный фундамент Python для:
- backend-разработки
- построения AI-агентов
- работы с данными
- интеграции с API

## Что уже реализовано

### Базовые конструкции
- Переменные и типы данных
- Арифметические операции
- Форматирование вывода

### Контроль потока
- Условия (if / elif / else)
- Обработка edge cases
- Валидация входных данных

### Циклы
- Обход списков
- Вычисление агрегатов
- Поиск максимального значения

### Агрегация данных
- Подсчёт total cost / revenue / profit
- Расчёт среднего значения
- Защита от деления на 0

### Фильтрация данных
- Исключение некорректных записей
- Работа с continue


## План развития

- Работа со списками глубже (фильтрация, создание новых коллекций)
- Функции и модульность
- Работа с файлами (CSV, JSON)
- HTTP-запросы
- Мини backend-сервис

---

Проект ведётся в формате ежедневной инженерной практики.
